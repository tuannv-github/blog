==========
Disk usage
==========

Estimate file space usage

.. list-table::

    * - Options
      - Shorten
      - Description
    * - :code:`--human-readable`
      - :code:`-h`
      - print sizes in human readable format (e.g., 1K 234M 2G)
    * - :code:`--summarize`
      - :code:`-s`
      - display only a total for each argument

.. code-block:: 

    du -h .
    du -hs .
    du -hs *
